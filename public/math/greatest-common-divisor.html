What is a greatest common divisor?

A greatest common divisor is the largest number that is able to divide every single member of a set of numbers.

It can be determined by using the Euclidean algorithm. The rules below need to be repeated until the first one is true.

a = b => GCD = a a > b => a = a - b b > a => b = b - a

Then the process is repeated by using the value of GCD and the next number in the set.
