What is a least common multiple?

A least common multiple is the smallest number divisible by every number in a set of numbers.

It can be determined by using the greatest common divisor.

LCM(a, b) = (a × b) ÷ GCD(a, b) Example: LCM(9, 3) = (9 × 3) ÷ GCD(9, 3) = 27 ÷ 3 = 9

This is because the LCM is equal to each of the numbers' unique factors multiplied by their common factor.

Then the process is repeated by using the value of LCM and the next number in the set.
